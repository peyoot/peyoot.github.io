文本换行：结尾两个空格
文本折叠：
<details>
<summary>点击这里查看隐藏内容</summary>

这里是被折叠隐藏的内容，可以是文字、代码块、列表等。
- 项目一
- 项目二


```javascript
console.log('也可以隐藏代码');
```

</details>

<details class="custom-details">
<summary>这里是标题</summary>

这里是正常显示的内容。
</details>

<style>
/* 隐藏默认小三角图标 */
.custom-details summary::-webkit-details-marker {
  display: none;
}
.custom-details summary::marker {
  display: none;
}

/* 自定义图标 */
.custom-details summary {
  position: relative;
  padding-left: 30px;
  cursor: pointer;
}

.custom-details summary::before {
  content: "🔍"; /* 这里可以换成任意图标：▶ ▼ + - 📖 等 */
  position: absolute;
  left: 0;
  font-size: 18px;
  transition: transform 0.3s ease;
}

/* 展开状态时改变图标 */
.custom-details[open] summary::before {
  content: "📖"; /* 展开时的图标 */
  /* 或者旋转图标 */
  /* transform: rotate(90deg); */
}
</style>

💡 使用技巧
组合使用：将多个 Emoji 组合可以创造更丰富的视觉效果，如 🌸✨💖 或 🌙🌟🦋
点缀文本：在文字中插入 Emoji 可以增加趣味性，如“今天学习了新知识 📚✨”
分类标记：用不同 Emoji 标记笔记分类，如 📝 表示笔记，🔍 表示研究
📋 快速复制列表
