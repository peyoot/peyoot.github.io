左上角：1400欧，同样，需离电极保持3mm以上的距离

触碰范围尺寸，即X，Y电极包裹的内尺寸为：225.32 X 127.88 mm， 但根据上面实测，各要减6mm，因此实际有效尺寸为219 x 121mm

为了保证线性，建议在触屏有效尺寸之外不放触控键。
