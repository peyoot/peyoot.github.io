# 开发板周边器件
VCC端口10uF+100nF，IOVDD端口10uF+100nF，X+，X-，Y+，Y-，电容对地分别470pF，串接10ohm。
SPI端口串接10ohm。
PENIRQ中断上拉4.7K。
SPI端口上拉电阻2.7K。

# 电阻屏的工作原理
X+，X-之间，以及Y+，Y-间测电阻,这是测量X-plate或Y-plate欧姆值，用于ti,x-plate-ohms参数。

Y方向，Y+和Y-之间构成通路，测X+同Y+之间的电阻分压值，
同理，当X+，X-构成通路，Y+同X+之间的电阻分压值。
用万用表可以简测X+，Y+间的电阻值：
未触碰时5.3M欧，相当于断路；

右上到右下角方向：
在右上角测得最小值在750欧左右，相当于接近X面板电阻。 须注意，离电极3mm内是无效的（即很难触碰出数据），所以触碰按钮不可放在这一区域。
右下角：520欧 左右

左上角到左下角方向
左上角：1400欧，同样，需离电极保持3mm以上的距离
左下角：1230区左右
