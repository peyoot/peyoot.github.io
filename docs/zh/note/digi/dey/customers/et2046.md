# 开发板周边器件
VCC端口10uF+100nF，IOVDD端口10uF+100nF，X+，X-，Y+，Y-，电容对地分别470pF，串接10ohm。
SPI端口串接10ohm。
PENIRQ中断上拉4.7K。
SPI端口上拉电阻2.7K。

# 电阻屏的工作原理
X+，X-之间，以及Y+，Y-间测电阻,这是测量X-plate或Y-plate欧姆值，用于ti,x-plate-ohms参数。
X+和X-间：X面板电阻700  ，触控引脚实测720欧
Y+和Y-间：Y面板电阻245  ，触控引脚实测270欧


Y方向，Y+和Y-之间构成通路，测X+同Y+之间的电阻分压值，但电压是芯片打开通道才能测的，平时可检测电阻
未触碰时5.3M欧，X+同Y+相当于断路；
没上电前四个方向触碰电阻值为：
左上角Y对应触碰电阻，1380欧， 左侧中间1450欧        左下角Y对应触碰电阻， 1200欧
中上角Y对应触碰电阻：1220欧   中中 1090             中下：980
右上： 890                  右中  800              右下 600

同理X方向，当X+，X-构成通路，Y+同X+之间的电阻分压值，对加电压来说方向变了，但没上电前，电阻的测法同X方向是一样的
